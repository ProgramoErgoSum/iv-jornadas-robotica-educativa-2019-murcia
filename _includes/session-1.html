# Internet de las Cosas Programación de un **servidor web en Python** para controlar desde el móvil una serie de LEDs conectados a los pines GPIO de nuestra Raspberry Pi. ![](images/preview.gif)
## Materiales - Raspberry Pi con SO Raspbian - Periféricos (monitor, teclado, ratón) - Conexión WiFi - Electrónica - 3 LEDs (amarillo y verde) - 3 resistencias - 4 cables (macho-hembra) - 1 protoboard
## Montaje Conectamos los LEDs a los pines GPIO 17 y 18 (BCM). ![](images/fritzing.png)
## Descargar código Vamos a descargar y ejecutar el código del servidor web para no tener que copiarlo. ```sh pi@raspberrypi:~ $ cd Desktop pi@raspberrypi:~/Desktop $ wget http://bit.ly/CPRMURCIA-IOT pi@raspberrypi:~/Desktop $ unzip master.zip pi@raspberrypi:~/Desktop $ python3 taller-iot-master/web/index.py ```
## Servidor web Vamos a probar el funcionamiento del servidor web accediendo a la dirección `localhost:8080` desde el navegador de nuestra Raspberry Pi. ![](images/navegador.png)
## Acceder desde el móvil Vamos a acceder a la IP de nuestra Raspberry Pi `xxx.xxx.xxx.xxx:8080` desde un dispositivo conectado a la misma red local. ```sh pi@raspberrypi:~ $ ifconfig wlan0: flags=4163 mtu 1500 inet xxx.xxx.xxx.xxx netmask 255.255.255.0 broadcast xxx.xxx.xxx.xxx ... ```
## Ejercicios propuestos - Accede desde diferentes dispositivos conectados a la misma red. - Comparte tu URL con el resto de compañeros. - Añade un LED de color rojo a la práctica y comprueba el correcto funcionamiento.
